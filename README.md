# Micropython Projects

## UBX vs NMEA
- UBX has better acccuracy than NMEA. 
    - Ublox calculates location and stored in UBX-NAV-PVT. 
    - NMEA output is generated by converting UBX-NAV-PVT.
    - Float point precision is reduced during conversion.
